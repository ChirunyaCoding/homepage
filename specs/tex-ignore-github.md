# タスク仕様: GitHub に .tex を公開しない

## 背景
- リポジトリに `.tex` ファイルを含めたくない。

## スコープ
- Git で `.tex` ファイルを追跡対象外にする設定を追加する。
- 既存の追跡状態に `.tex` がないか確認する。

## 制約
- 既存の他設定を壊さない。
- 変更は最小限にする。

## 受け入れ条件
- `.gitignore` に `.tex` を無視するルールが存在する。
- `git ls-files "*.tex"` の結果が空であることを確認できる。

## 非対象
- LaTeX ビルド環境の構築。
- 既存コミット履歴からの `.tex` 完全削除（履歴改変）。
